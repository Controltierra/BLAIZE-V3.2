## Descarga del Ejecutable

El archivo .exe es demasiado grande para GitHub (>100MB). 

**Para obtener el ejecutable:**
1. Ve a la carpeta local: `windows-amd64/Blaize_V3_Modificado.exe`
2. O descárgalo desde: [Releases](https://github.com/Controltierra/Blaize-V3-Modified/releases)

**Para compilar tú mismo:**
1. Abre `Blaize_V3.pde` en Processing 3
2. File  Export Application  Windows 64-bit + Embed Java
3. El .exe se creará en `windows-amd64/`
